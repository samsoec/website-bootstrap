# Troubleshooting

Step-by-step problem solving.

## General Troubleshooting Steps

1. **Refresh the page**
2. **Clear browser cache**
3. **Try different browser**
4. **Check internet connection**
5. **Log out and back in**
6. **Contact support**

## Browser Issues

### Recommended Browsers
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

### Clear Cache

**Chrome:**
1. `Cmd+Shift+Delete` (Mac) or `Ctrl+Shift+Delete` (Windows)
2. Select "Cached images and files"
3. Click "Clear data"

**Safari:**
1. Safari → Preferences → Privacy
2. Click "Manage Website Data"
3. Remove all

## Login Issues

### Forgot Password
1. Click "Forgot password"
2. Enter email
3. Check inbox
4. Follow reset link

### Account Locked
Contact administrator after multiple failed attempts.

### Session Keeps Expiring
- Disable browser extensions
- Check cookie settings
- Try incognito mode

## Performance Issues

### CMS Loading Slowly

**Fixes:**
- Close unused browser tabs
- Clear browser cache
- Check internet speed

### Content Taking Long to Save
- Reduce image sizes
- Save more frequently
- Check for large rich text content

## When to Contact Support

Contact technical support when:
- Errors persist after troubleshooting
- You're locked out
- Data appears missing
- Features don't work
- Server errors appear

  - Time of occurrence

## Emergency Procedures

### Accidentally Deleted Content
1. Stop immediately
2. Contact developers
3. Database backups may help
4. Document what was deleted

### Major Display Issues
1. Don't make more changes
2. Screenshot the problem
3. Contact support
4. Wait for fix before continuing

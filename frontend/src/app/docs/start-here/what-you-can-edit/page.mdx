# What You Can Safely Edit

import { Callout } from 'nextra/components'

Here's what you can confidently change without breaking anything.

## Safe to Edit

| What | Where to Find It |
|------|------------------|
| Page text & headings | Content Manager → Pages |
| Images & videos | Media Library |
| Blog posts | Content Manager → Articles |
| Navigation menu | Content Manager → Global |
| Footer content | Content Manager → Global |
| SEO titles & descriptions | Each content entry's SEO section |
| Section order on pages | Drag sections in the page editor |

## How Pages Are Built

Each page is made of **sections** stacked together like building blocks — the same sections you see rendered on the website map to entries in the "Content Sections" dynamic zone in the CMS.

```
About Page
├── Hero (banner at top)
├── Feature Rows
└── Testimonials
```

<Callout type="info">
You can reorder sections by dragging them. The order in the CMS matches the order on the website.
</Callout>

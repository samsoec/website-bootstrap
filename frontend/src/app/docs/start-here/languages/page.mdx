# Add Translations

import { Callout, Steps } from 'nextra/components'

The website is configured for **English, German, and Czech** (see `i18n-config.ts` in the frontend). Not every content type is localized — check with your developer which ones are.

## How to Add Another Language Version

<Steps>
### Open the content

Find and open the content you want to translate.

### Click "Locales" in the sidebar

Look for the language switcher on the right side (only shown for content types that support locales).

### Select the target language

The page structure stays the same — just translate the text fields.

### Publish separately

Click Publish for each language version.
</Steps>

<Callout type="warning">
Publishing one language does **not** auto-publish the others. Each language publishes separately.
</Callout>

## What to Translate

- All text content
- SEO title & description
- Button labels

## Tips

- **Check the globe icon** in content lists to see which items have translations
- If a locale shows the default-locale content, make sure that locale's version is published
